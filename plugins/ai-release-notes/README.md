# Ai-Release-Notes Plugin



## Installation

This plugin is not included with the `auto` CLI installed via NPM. To install:

```bash
npm i --save-dev @auto-it/ai-release-notes
# or
yarn add -D @auto-it/ai-release-notes
```

## Usage

```json
{
  "plugins": [
    "ai-release-notes"
    // other plugins
  ]
}
```
